スマホ修理受付管理アプリ

Javaの勉強を始めて作ったプログラムです。
前職のiCrackedで4年半スマホ修理をしていたので、その現場で使えるような「受付システム」をイメージして作りました。

このアプリを作った理由
学習内容をアウトプットするため。自分が実際にやっていた仕事にしたほうが、何を作っているのか理解しながら書けると思ったからです。

今の自分にできること
今はまだ基礎の基礎ですが、以下の機能を実装しました。
・名前と機種名を打ち込んで保存する機能
・保存したものを一覧で表示する機能
・0番を押してアプリを終わらせる機能

苦労したところ
最初は1回入力したらプログラムが終わってしまっていたのですが、while文を使って「0を押すまで何度も繰り返す」ようにするのが難しくて、何度もエラーを出して直しました。

 これからやりたいこと
・今はアプリを閉じるとデータが消えてしまうので、次はデータベースに保存できるようにしたいです。
・もっと読みやすいコードが書けるようになりたいです。
・削除機能をつける

 開発環境
